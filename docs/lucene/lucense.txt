Apache Lucene
------------------------
The indexing and search library for a high performance, full-text search engine

Open source, free to use written in Java, ported to other languages

Just like Hadoop in the distributed computing world, Lucene is the nucleus גרעין of several technologies
built around it:

Solr -A search server with  distributed indexing,load balancing, replication, automated recover, centralized configuration
Apache Nutch - Web crawling and index parsing
createDB- Open source, SQL distributed database
And elasticsearch...
